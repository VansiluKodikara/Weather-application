# 🌦️ WeatherFinder

A simple and clean weather web application **currently under development**.

Built with HTML, CSS, and vanilla JavaScript. It will show real-time weather information for any city.

## 🚧 Current Status

**In Development** — The project is still being built.  
More features and improvements are coming soon!

## ✨ Planned Features

- Search weather by city name
- Display current temperature, feels like, humidity, wind speed & condition
- Responsive design for mobile and desktop
- Clean and modern UI
- Real-time weather data using a weather API

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
